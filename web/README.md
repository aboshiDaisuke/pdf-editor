# PDF Editor — ブラウザ版

Python サーバー（Flask / PyMuPDF）を**廃止**し、**ブラウザ内だけで完結**する PDF 編集アプリです。
PDF はあなたの PC の外に出ず、すべてブラウザの中で処理されます（アップロード無し・ネット不要）。

裏側のエンジンはデスクトップ版と同じ **MuPDF**（の公式 WebAssembly 版 `mupdf.js`）を使っているため、
本文の直接編集・注釈・検索などの機能をほぼそのまま再現しています。

## 動かし方（ローカル）

ES モジュールと WASM を使うため、**HTTP 経由で開く必要があります**（`index.html` を `file://` で直接開くと動きません）。
`web/` フォルダを任意の静的サーバーで配信してください。

```bash
cd web
python3 -m http.server 8000      # もしくは:  npx serve .
```

ブラウザで <http://localhost:8000> を開く。

## 公開（デプロイ）

`web/` フォルダ一式を任意の静的ホスティングに置くだけです（ビルド不要）。

- **GitHub Pages** … リポジトリの Pages 設定で `web/` を公開、または `web/` の中身をルートに配置
- **Vercel / Netlify / Cloudflare Pages** … 出力ディレクトリを `web/` に指定
- 社内の任意の静的 Web サーバー

> WASM はストリーミングコンパイルを使わないので、`.wasm` の MIME 設定は不要です。

## ファイル構成

| ファイル | 役割 |
|---|---|
| `index.html` | UI 一式（HTML/CSS/JS）。デスクトップ版と同じ見た目＋ライト/ダーク切替 |
| `pdf-engine.js` | `mupdf.js` をラップした PDF エンジン（旧 `server.py` 相当を全面再現） |
| `mupdf.js` / `mupdf-wasm.js` / `mupdf-wasm.wasm` | MuPDF 公式 WebAssembly ランタイム |

## 使い方

- **開く**: 「開く」ボタン、またはページ上へ PDF をドラッグ&ドロップ
- **編集**: 文字（ダブルクリックで直接編集 / 追加）・画像・図形・注釈・ページ整理・検索・フォーム入力
- **保存**: 「保存」「別名で保存」→ 編集済み PDF を**ダウンロード**
- **テーマ**: ヘッダー右端の 太陽/月 アイコンで **ライト⇄ダーク** 切替（選択はブラウザに記憶）

## デスクトップ版との違い

| | デスクトップ版 (`server.py` + pywebview) | ブラウザ版 (`web/`) |
|---|---|---|
| 実行 | `python server.py` | 静的配信して開くだけ |
| ファイル選択/保存 | OS ネイティブダイアログ | ブラウザの選択 / ダウンロード |
| 上書き保存 | 元ファイルに上書き | ダウンロード（元ファイル上書きはブラウザ制約のため非対応※） |
| 配布 | `.app`（PyInstaller） | URL を共有するだけ |

※ Chrome/Edge の File System Access API を使えば将来的に上書き保存も追加可能です。
